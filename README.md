# nltk-notebook


## initial setup
  - use virtualenvwrapper:
    - `mkvirtualenv nltk-notebook`
  - install packages:
    - `pip install -r requirements.txt`
  - install nltk stuff:
    - `python -m nltk.downloader popular`
## working

- use virtualenvwrapper:
  - `workon nltk-notebook`
- start jupiter laboratory:
  - `jupyter-lab`
